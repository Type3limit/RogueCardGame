## 职业概览

| 属性 | 数值 |
|------|------|
| HP | 85 |
| 核心机制 | **超载 (Overcharge)** — 攻击牌累积层数，花费层数释放强力效果 |
| 卡池比例 | 近战 60% / 远程 15% / 通用 25% |
| 默认站位 | 前排 |
| 设计幻想 | 冲锋陷阵的突击兵 / 不可撼动的钢铁堡垒 / 精确打击的战术指挥官 |

## 核心机制：超载 (Overcharge)

打出攻击牌时累积超载层数。超载层数本身没有被动效果——你需要通过特定卡牌"消耗超载"来释放大伤害或强力效果。

- 每张攻击牌 → +1 超载层数
- 超载层数在战斗内持续存在（不会自然消退）
- "消耗超载"效果会清空所有层数，效果按层数缩放

**核心张力**：你是现在就消耗超载打伤害，还是继续积攒等待更大的爆发？

## 核心植入体（核心插槽）

| 植入体 | 规则改变 | 定义的流派 |
|--------|---------|-----------|
| **超载核心**（默认） | 标准超载机制 | 蓄力爆发型 |
| **铁壁核心** | 取消超载。前排时每回合开始获得"上回合近战牌数 ×4"护甲 | 前排坦克型 |
| **狂战核心** | 取消超载。每张攻击牌让后续攻击牌 +2，后排远程牌叠加翻倍为 +4 | 连锋输出型 |
| **指挥核心** | 取消超载。前排通用牌 +3 护甲；后排通用牌额外抽 1 牌 | 双模式灵活型 |

## 六大构筑流派

### 流派一：超载爆发 (Overcharge Burst)

**核心组合**：超载核心 + 标准神经接口
**站位策略**：常驻前排
**幻想**：蓄力 → 一击必杀。像给枪上膛一样，每一拳都在蓄力，然后一发超载释放炸开整个战场。

#### 玩法概述

最经典的先锋玩法。通过大量攻击牌累积超载层数，然后用"超载释放"类卡牌一次性消耗全部层数打出超高爆发伤害。整个回合的节奏是"蓄 → 蓄 → 蓄 → 爆"。

#### 核心策略

- **前期**：快速用低费攻击牌堆叠超载，第 2-3 回合打出第一次超载释放
- **中期**：寻找"超载共振"等关键稀有牌，让超载释放伤害倍增
- **后期**：追求一回合 5+ 超载层然后释放 = 秒杀精英级伤害

#### 关键卡牌

- **超载释放** (核心输出) + **连续突刺** (快速叠层) + **蓄能冲拳** (叠层+防御)
- **超载共振** (稀有) — 消耗超载时，每层额外 +3 伤害
- **能量回馈** — 消耗超载后回复 1 能量，允许继续行动

#### 站位考量

几乎永远站前排——60% 的牌是近战，前排才能全额使用。偶尔在需要躲避大招时退后排一回合。

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 爆发天花板极高，单回合伤害全职业最强 | 蓄力期间输出真空，面对群怪压力大 |
| 构筑思路简单直观，新手友好 | 前排常驻吃很多伤害，依赖护甲和 HP |
| 对精英/Boss 极其有效 | 超载释放被冻结（低温环境）时节奏崩盘 |

### 流派二：铁壁堡垒 (Iron Fortress)

**核心组合**：铁壁核心 + 标准神经接口
**站位策略**：锁死前排
**幻想**：移动的钢铁堡垒。你不是在打怪，你是在让怪打自己——然后用积攒的护甲砸回去。

#### 玩法概述

铁壁核心让你每回合根据上回合打出的近战牌数量自动获得护甲（×4）。核心循环：站前排 → 打近战牌 → 下回合自动获甲 → 用"护盾猛击"等卡牌将护甲转化为伤害。

#### 核心策略

- **前期**：确保每回合至少打 3+ 张近战牌（= 下回合 12 护甲），靠自然护甲硬吃伤害
- **中期**：寻找"护盾猛击"和"铁甲反击"——将叠高的护甲转化为输出
- **后期**：配合"钢铁意志"（护甲不在回合结束时消失），实现永久高护甲状态

#### 关键卡牌

- **护盾猛击** (护甲→伤害) + **连环挥砍** (多次近战=高叠甲) + **钢铁意志** (护甲保留)
- **反击架势** — 前排被攻击时反弹护甲值 30% 的伤害
- **要塞化** (传说) — 本回合护甲翻倍，但你不能换位

#### 站位考量

**永远不离开前排**。离开 = 失去护甲来源 = 整个 build 崩溃。最"站位锁定"的流派。

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 生存能力全流派最强 | 对绕过护甲的 AOE 非常脆弱 |
| 护甲转伤害输出稳定 | 完全无法处理后排敌人 |
| 天然克制单体高伤害敌人 | 敌人强制换位技能会毁掉节奏 |

### 流派三：狂战风暴 (Berserker Storm)

**核心组合**：狂战核心 + 脉冲处理器
**站位策略**：常驻后排
**幻想**：站在后排像机关枪一样扫射——每一发比上一发更痛。火力永不停息，越打越疯。

#### 玩法概述

狂战核心让每张攻击牌给后续攻击牌 +2 伤害（可叠加），后排远程牌叠加翻倍为 +4。配合脉冲处理器（打牌抽牌），你可以在后排用大量低费远程攻击牌打出雪崩式伤害。

#### 核心策略

- **前期**：积极寻找 0-1 费远程攻击牌，构筑"低费轰炸"牌组
- **中期**：脉冲处理器的"打一抽一"让你每回合打出 7-10 张牌，狂战叠加变得恐怖
- **后期**：最后几张攻击牌每张可以附加 +16 ~ +20 额外伤害

#### 关键卡牌

- **速射** (0费远程) + **弹幕** (1费打2次) + **充能射击** (打完抽牌)
- **弹药补给** — 将 1 张手牌转化为 2 张随机远程攻击牌
- **战争狂热** (传说) — 本回合每打出一张攻击牌额外 +1 费用上限

#### 站位考量

常驻后排获得翻倍叠加。牌组中几乎不应该有近战牌——完全颠覆先锋"近战为主"的默认印象。

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 单回合总输出极高（累积伤害） | 需要大幅改造卡池（默认60%近战不能用） |
| 后排安全，几乎不吃伤害 | 脉冲处理器每回合只抽3牌，前几张无加成 |
| 对群怪效果极佳（多次攻击分散目标） | 穿透敌人（打后排）时极其脆弱 |

### 流派四：战术指挥 (Tactical Commander)

**核心组合**：指挥核心 + 战场接口
**站位策略**：每回合换位
**幻想**：你是战场的棋手，不是棋子。前排站一下攒甲，后排跳一下抽牌，每一步都是精确计算。

#### 玩法概述

指挥核心让通用牌在前后排有不同效果（前排+3甲，后排+1抽牌）。战场接口让换位免费且换位后下一张牌 -1 费。两者结合：你每回合都在前后排之间跳转，最大化双模式收益。

#### 核心策略

- **标准节奏**：开局前排 → 打 1-2 张通用牌攒甲 → 换位到后排（免费+减费）→ 打通用牌抽牌 → 用抽到的牌输出
- **灵活切换**：根据敌人行动意图决定本回合最终停在前排还是后排
- **后期**：追求大量通用牌 + 少量高效攻击牌的牌组结构

#### 关键卡牌

- **战术撤退** (换位+抽牌) + **指令广播** (通用,全体+2伤害) + **相位突进** (位置反应牌)
- **前线部署** — 0费通用牌，换到前排并获得 5 护甲
- **全局视野** (传说) — 本回合你的通用牌同时触发前排和后排效果

#### 站位考量

**频繁换位是核心**。不换位这个 build 就不存在。需要精确规划每回合的换位时机和出牌顺序。

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 极度灵活，攻防兼备 | 操作复杂度全流派最高 |
| 任何敌人配置都有应对方案 | 没有一个维度特别突出 |
| 通用牌不受站位限制，利用率高 | 多人模式回合时间长 |

### 流派五：过载战机 (Overload Fighter)

**核心组合**：超载核心 + 过载回路
**站位策略**：前排为主，偶尔后排
**幻想**：用光每一点能量的精密战争机器。没有浪费，没有余裕——要么完美，要么自爆。

#### 玩法概述

过载回路给你 4 能量（比标准多 1），但未使用的能量在回合结束时变成等量自伤。配合超载核心：更多能量 → 更多攻击牌 → 更快蓄超载，但必须精确用完。

#### 核心策略

- **牌组构筑**：严格控制费用曲线，确保每回合恰好用完 4 能量
- **黄金配比**：4×1费 / 2×1费+1×2费 / 1×1费+1×3费 = 恰好4
- **保险措施**：携带"能量转甲"类牌，将多余能量转化为护甲

#### 关键卡牌

- **精确打击** (1费高效攻击) + **双重斩** (2费打两次) + **能量转甲** (剩余能量→护甲)
- **过载释放** — 消耗超载时，若本回合恰好用完所有能量，伤害 ×1.5
- **完美校准** (传说) — 被动：回合结束时若剩余能量为 0，抽 2 牌

#### 站位考量

主要站前排（充分利用4能量打近战牌），手牌不完美时退后排用远程+通用牌消耗能量避免自伤。

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 4能量 = 行动力多33% | 容错率极低，费用不匹配就自伤 |
| 超载蓄力速度也快33% | 受低温/等离子风暴环境严重影响 |
| 精确构筑带来强掌控感 | 新手不友好，需理解费用曲线 |

### 流派六：相位游击 (Phase Guerrilla)

**核心组合**：超载核心 + 战场接口
**站位策略**：激进换位
**幻想**：幽灵突击队——冲到前排砍一刀，闪回后排。敌人永远抓不到你，但你的刀永远够得到他们。

#### 玩法概述

战场接口让换位免费且换位后首张牌 -1 费。超载核心让攻击牌叠超载。流程：后排开局 → 免费换前排 → 减费近战牌叠超载 → 打完退后排 → 重复。

#### 核心策略

- **打了就跑**：每回合在前排打 2-3 张减费近战牌，然后退回后排保命
- **位置反应牌**："相位突进"完美契合（前排伤害退后排 / 后排进前排获甲）
- **超载释放时机**：前排蓄满超载，切后排安全释放

#### 关键卡牌

- **相位突进** (位置反应) + **闪击** (攻击后退后排) + **幽灵步** (0费,换位+隐蔽)
- **残影** — 每次换位后，对所有前排敌人造成 3 伤害
- **时空裂隙** (传说) — 本回合每次换位后下一张攻击牌伤害翻倍（仅一次）

#### 站位考量

**疯狂换位**。一回合可能换位 2-3 次。核心是"前排输出 → 后排躲攻击"节奏。

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 攻防兼备，前排输出后排保命 | 出牌顺序极其重要，回合思考时间长 |
| 换位触发多种额外效果 | 牌组需要混合近战/远程/通用 |
| 机动应对任何敌人配置 | "换位后失去护甲"环境直接崩盘 |

## 与其他职业的定位区分

- **vs 灵能者**：先锋靠物理攻击叠加，灵能者靠共鸣增幅。先锋更直接暴力，灵能者更需要铺垫。
- **vs 黑客**：先锋正面输出，黑客 debuff 控制。先锋靠"打死"，黑客靠"削弱"。
- **vs 共生体**：先锋的资源是能量+超载（外部），共生体的资源是 HP（内部）。先锋更安全，共生体更极端。
