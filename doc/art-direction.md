# 美术方向与资源规范

## 当前目标

Phase 4 的美术优先级先从“职业身份”开始：四个可选职业必须在主菜单、战斗立绘、卡面底图中拥有清晰、一眼可辨的视觉身份。游戏整体以日本二次元动漫 / 卡牌手游立绘为基准，后续每张卡的独立插画按职业与机制批量替换。

## 世界观视觉基调

- 时间：2187 年，巴别塔沉默四十年后重新发出“进度查询”。
- 场景：近地轨道实验站异化成的巴别塔，金属残骸、非欧空间、数据/灵能/生物质三种异常层叠。
- 总体风格：日系二次元动漫 + 赛璐璐阴影 + 精细线稿 + 卡牌手游立绘；保留成熟赛博朋克、轨道废墟与身体异化主题，但角色表现必须是 2D anime / gacha RPG illustration，不走照片写实、欧美写实概念图或 3D 渲染感。
- 画面约束：角色优先，五官与发型要有动漫辨识度；材质可以精细，但不要真实摄影皮肤、写实镜头景深、电影海报式厚涂。
- 清洁度约束：图像必须低噪点、低颗粒、少碎屑、少粒子、少微纹理；角色本体也要保持干净，用大块装甲面、清晰布料阴影、少量可读发束和受控发光点表现材质，避免细小划痕、密集机械线、斑点高光和脏污纹理。
- 动作约束：职业主视觉不能是四个角色同样站姿；每个角色要有不同动作线和重心，至少包含前压、悬浮、突进、扭转等差异化姿态，让缩到卡面里仍然能读出职业性格。

## 四职业主视觉

### 先锋 / Vanguard

- 核心：铁幕工业重型动力装甲，前排堡垒，超载反应堆。
- 颜色：军灰、磨损金属银、警示橙、过载炽白。
- 记忆点：前压防御或冲锋姿态、超大右臂义肢、装甲接缝热光、脊柱散热鳍片、等离子刃/动力拳套。

### 灵能者 / Psion

- 核心：女性普罗米修斯计划幸存者，脆弱身体与危险意识场的反差。
- 颜色：暗紫、靛蓝、青白神经光。
- 记忆点：女性动漫角色轮廓、悬浮姿态、伸手牵引重力的动作线、悬浮增幅环、白色发光双眼、面部神经纹路、少量大块漂浮碎片。

### 黑客 / Netrunner

- 核心：女性影线网络数据战士，协议栈与入侵界面；气质应自信、轻佻、游刃有余，像在敌人意识到之前已经完成入侵。
- 颜色：黑色战术服、冷金属、青色数据光。
- 记忆点：女性动漫角色轮廓、轻松侧身操控界面的姿态、挑衅式微笑、AR 视镜或神经接口、外接接口背包、线缆/数据流、模块化义体。
- 材质要求：衣装以哑光黑色战术布料和大块义体面为主，只保留少量青色边光；避免亮面紧身衣的碎反光、密集高光点、细划痕和金属噪点。

### 共生体 / Symbiote

- 核心：女性塔源活性生物质宿主，人体与异种生命共同作战；重点是“被寄生强化后的战斗感”，不是完全怪物化。
- 颜色：黑绿生物质、毒性荧光、湿润有机光泽。
- 记忆点：银白 / 灰绿短发或不对称短发、扭转捕食姿态、后颈根植点、非对称寄生肩甲、巨型有机爪臂、少量粗线条触手弧线、非对称生物装甲。
- 避免项：不能使用黑发高马尾、黑色战术黑客轮廓、AR 视镜或数据线主视觉；绿色发光只保留少量大块缝线，避免满身细裂纹和随机荧光噪点。

## 资源接入规则

- 职业主立绘：`resources/textures/characters/<class>.png`
- 原始生成图集：`resources/textures/characters/source/phase4_class_lineup.png`
- 战斗动态立绘：`resources/textures/characters/animations/<class>/<action>/frame_00.png` 起的 PNG 帧序列。
- 战斗动态立绘必须是透明背景 PNG，玩家角色位于战斗画面左侧时必须朝右、面向敌人；不能直接复用面朝镜头的职业选择主视觉。
- 动作帧标准：每个战斗动作至少 25 张透明 PNG，对应 25fps / 1 秒；待机使用 1 秒循环，攻击 / 加甲 / 职业特效为一次性序列，播放完回到待机。
- 每帧必须有可读的姿态、重心、浮动、能量或寄生体变化，不能只做近似相同的亮度闪烁；自动验收会检查 `frame_00.png` 与 `frame_24.png` 不完全相同。
- 当前 Godot UI 层优先使用 PNG 帧序列，由 C# 在 `TextureRect` 上以 25fps 轮播；不把 GIF 作为动态立绘资源格式。
- 当前动作目录：`combat_idle`、`attack`、`gain_armor`，以及职业核心特效目录：先锋 `overload`、灵能者 `resonance`、黑客 `protocol_stack`、共生体 `erosion`。
- 卡牌独立插画：`resources/textures/cards/art/<class>/<card_id>.png` 或 `.webp`
- 卡牌插画不应包含卡框、标题、费用、稀有度或 UI 文本，只做 art stage 中的画面层。
- 当前卡面会同时使用职业人物底图和卡牌 `artPath` 前景层；后续替换单卡插画时不需要改 UI 代码。

## 首批生成 Prompt 摘要

日本二次元赛博朋克卡牌手游风格的四职业等宽竖版图集，一图四栏，无文字无 UI 框。每栏分别表现前压姿态的 Vanguard、悬浮伸手的女性 Psion、自信轻佻操控界面的女性 Netrunner、扭转捕食姿态的 Symbiote，在巴别塔内部环境中保持统一 anime/cel-shaded 画风、统一比例、不同职业色彩。画面要求 clean image、low visual noise、large readable shapes、smooth color blocks、different dynamic poses。明确排除 photorealistic、western realistic concept art、3D render、grain、dust speckles、particle clouds、grunge texture、tiny scratches、noisy micro highlights。
