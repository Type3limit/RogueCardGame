# 灵能者 (Psion) — 职业设计与流派指南

> 本文档应位于 `doc/roleDesign/灵能者.md`。卡牌详细列表见 `灵能者卡牌.md`。

---

## 职业概览

| 属性 | 数值 |
|------|------|
| HP | 65 |
| 核心机制 | **共鸣 (Resonance)** — 打出通用牌累积共鸣层数，共鸣层数增幅伤害效果 |
| 卡池比例 | 近战 10% / 远程 50% / 通用 40% |
| 默认站位 | 后排 |
| 设计幻想 | 蓄力释放的灵能大师 / 精准标记的念力猎手 / 打破常规的越境战士 |

## 核心机制：共鸣 (Resonance)

打出通用牌（技能牌）时累积共鸣层数。共鸣增幅你的下一次伤害。

- 每张通用牌 → +1 共鸣层数
- 默认规则（共鸣核心）：每层共鸣让下一次伤害 +2，造成伤害后消耗所有共鸣
- 共鸣在战斗内持续存在直到被消耗

**核心张力**：你要花多少回合铺垫（打通用牌攒共鸣）vs 直接输出？铺垫越多一击越猛，但铺垫期间你没有伤害输出。

## 核心植入体

| 植入体 | 规则改变 | 定义的流派 |
|--------|---------|-----------|
| **共鸣核心**（默认） | 每层共鸣 +2 伤害，伤害后消耗全部共鸣 | 蓄力一击型 |
| **回响核心** | 共鸣不再在伤害后消耗，每回合末失去一半（向上取整），每层仅 +1 伤害 | 持续输出型 |
| **念动核心** | 取消共鸣。远程牌标记目标 1 回合，对已标记目标出牌费用 -1 | 标记集火型 |
| **越境核心** | 近战牌也累积共鸣（每张+2层），前排时共鸣伤害加成翻倍 | 前排玻璃炮 |

## 六大构筑流派

### 流派一：共鸣爆弹 (Resonance Bomb)

**核心组合**：共鸣核心 + 标准神经接口
**站位策略**：常驻后排
**幻想**：用整个回合铺垫灵能共鸣，然后一发"灵能核弹"把目标从存在中抹除。

#### 玩法概述

经典灵能者玩法。前几张打通用牌攒共鸣，最后一张远程攻击牌消耗全部共鸣打出巨额爆发。和先锋超载爆发的区别：先锋靠多张攻击牌叠超载，灵能者靠通用牌铺垫后一发终结。

#### 核心策略

- **标准回合**：3 张通用牌（3层共鸣）→ 1 张远程攻击牌（+6 伤害）
- **爆发回合**：用"冥想"等 0 费通用牌额外叠层，追求 5-7 层共鸣 → 一击 +10~14
- **构筑方向**：大量低费通用牌 + 少量高基础伤害的远程攻击牌

#### 关键卡牌

- **灵能冲击** (高基础伤害远程) + **冥想** (0费通用,+1共鸣) + **精神聚焦** (通用,+2共鸣)
- **共鸣增幅器** (稀有) — 消耗共鸣时每层 +3 而非 +2
- **灵能核弹** (传说) — 消耗所有共鸣，伤害 = 共鸣层数 × 8

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 单次爆发伤害极高 | 铺垫回合无伤害输出 |
| 后排安全 | HP 最低（65），遇穿透即危 |
| 通用牌多，可同时提供护甲/抽牌 | 对多目标效率低（一发只打一个） |

### 流派二：回响脉冲 (Echo Pulse)

**核心组合**：回响核心 + 脉冲处理器
**站位策略**：常驻后排
**幻想**：持续共鸣的能量脉冲——你不追求一击必杀，而是让每一发都带着灵能增幅，源源不断。

#### 玩法概述

回响核心让共鸣不再在伤害后消耗（但每回合末减半），每层 +1 伤害。配合脉冲处理器（打牌抽牌），你在一个回合中大量出牌，每张攻击牌都有共鸣加成，维持高共鸣输出。

#### 核心策略

- **维持节奏**：每回合至少打 2 张通用牌补充共鸣（抵消回合末减半）
- **输出窗口**：共鸣高（6-8层）时连续打远程攻击牌，每张 +6~8 伤害
- **脉冲引擎**：脉冲处理器打牌抽牌 → 抽到更多通用牌 → 补充共鸣 → 继续出牌

#### 关键卡牌

- **回响波** (1费远程,造成伤害后+1共鸣) + **能量涟漪** (0费通用) + **脉冲射线** (1费远程)
- **持续共振** (稀有) — 回合结束时共鸣只减少 1 层而非减半
- **共鸣风暴** (传说) — 本回合每次造成伤害后获得 1 层共鸣（而非消耗）

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 稳定持续输出，不依赖爆发窗口 | 单次伤害不如共鸣爆弹 |
| 多目标效率高（每张牌都有加成） | 共鸣维持需要持续打通用牌 |
| 脉冲引擎让回合行动数极多 | 脉冲处理器只抽3牌，前期手牌少 |

### 流派三：念动猎手 (Telekinetic Hunter)

**核心组合**：念动核心 + 预知模块
**站位策略**：后排为主
**幻想**：精准锁定一个目标，用念力标记它，然后连续低费攻击将它从战场上抹除。逐个击破的冷血猎手。

#### 玩法概述

念动核心取消共鸣，改为标记机制：远程牌命中目标后标记 1 回合，对已标记目标打牌费用 -1。预知模块让你看牌堆顶 2 张选 1 张，确保你总能拿到需要的牌。

#### 核心策略

- **标记 → 倾泻**：第一张远程牌标记目标 → 后续牌对该目标全部 -1 费 → 同费用打出更多牌
- **预知精准**：预知模块确保你每回合都能拿到远程牌（用来标记）或高伤害牌（用来收割）
- **逐个击破**：先杀后排高价值目标（治疗、强化者），再处理前排

#### 关键卡牌

- **念力锁定** (1费远程,标记+小伤害) + **精神穿刺** (2费远程,对标记目标+50%伤害)
- **猎手直觉** — 对已标记目标造成伤害时抽 1 牌
- **念力处决** (传说) — 对已标记且 HP < 30% 的目标直接消灭

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 单目标清除效率极高 | 对群怪效率低（只能标记一个） |
| 减费机制让能量极其高效 | 没有共鸣，缺乏伤害增幅 |
| 预知模块保证手牌质量 | 遇到高HP肉盾时节奏变慢 |

### 流派四：越境战士 (Boundary Breaker)

**核心组合**：越境核心 + 标准神经接口（或过载回路）
**站位策略**：常驻前排
**幻想**：打破"灵能者=后排"的刻板印象。你是用灵能包裹拳头的近战战士，站在最危险的前排，用翻倍共鸣打出全职业最高的瞬间爆发。

#### 玩法概述

越境核心让近战牌也能累积共鸣（每张+2层，比通用牌+1更快），且前排时共鸣伤害加成翻倍。你站前排用近战牌飞速叠共鸣，然后用远程/近战攻击牌释放翻倍伤害。

#### 核心策略

- **激进前排**：站前排，近战牌每张+2共鸣，2张近战就是4层→翻倍后+16伤害
- **高风险高回报**：65 HP 站前排非常危险，但翻倍共鸣伤害是全游戏最高的单次伤害
- **构筑改造**：需要增加近战牌比例（默认只有10%），事件和商店优先选近战牌

#### 关键卡牌

- **灵能拳** (1费近战,造成伤害+1共鸣) + **越境打击** (2费近战,+3共鸣)
- **灵能护体** (通用,获得护甲=当前共鸣层数) — 前排生存的保障
- **越境终结** (传说) — 消耗所有共鸣，伤害 = 共鸣层数 × 5（前排翻倍后 ×10）

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 理论爆发最高（翻倍共鸣） | 65HP 前排极度脆弱 |
| 共鸣叠加速度最快 | 需要大改卡池（增加近战牌） |
| 打破预期，敌人AI可能没针对 | 被强制换到后排就废了 |

### 流派五：精神风暴 (Psi Storm)

**核心组合**：共鸣核心 + 过载回路
**站位策略**：后排
**幻想**：危险的精神超载——4 能量让你铺垫更多共鸣，然后释放毁灭性的灵能风暴。但未用完的能量会反噬你。

#### 玩法概述

过载回路给 4 能量（多 1），配合共鸣核心，你有更多行动来叠共鸣和释放。但 65 HP 的灵能者承受未使用能量的自伤更加致命——你必须精确规划每一点能量。

#### 核心策略

- **4能量最优分配**：3张通用牌(3费,3层共鸣) + 1张远程攻击(1费) = 完美4能量
- **AOE 方向**：寻找"灵能风暴"等 AOE 卡牌，用高共鸣+AOE 一波清场
- **能量保险**：携带 0 费通用牌，在能量有余时用来叠共鸣+消耗能量

#### 关键卡牌

- **灵能风暴** (2费远程AOE) + **冥想** (0费通用) + **精神聚焦** (1费通用,+2共鸣)
- **超载冥想** — 未使用的能量不自伤，而是转化为等量共鸣层数
- **灭世波动** (传说) — 3费，消耗全部共鸣，对所有敌人造成 共鸣×4 伤害

#### 优劣势

| 优势 | 劣势 |
|------|------|
| AOE 清场能力最强 | 65HP+自伤=随时可能暴毙 |
| 4能量每回合行动更多 | 费用不匹配自伤格外致命 |
| 共鸣+AOE一波解决群怪 | 单目标效率不如念动猎手 |

### 流派六：预言编织 (Prophecy Weave)

**核心组合**：回响核心 + 预知模块
**站位策略**：后排
**幻想**：你已经看见了未来。每一步都在你的计算之中。你不追求最高伤害，而是追求每一张牌都在最完美的时机打出。

#### 玩法概述

预知模块让你每 2 回合看牌堆顶 2 张选 1 张，回响核心维持持续共鸣。这个 build 的核心不是爆发，而是**完美控制**——你总能拿到需要的牌，维持理想的共鸣层数和手牌质量。

#### 核心策略

- **牌堆操控**：预知模块选择最需要的牌，不需要的放回底部（不会再抽到）
- **共鸣维持**：回响核心 + 持续打通用牌 = 始终保持 4-6 层共鸣
- **长期碾压**：不追求一回合爆发，而是每回合稳定 +4~6 × 多张攻击牌的持续输出

#### 关键卡牌

- **预言射线** (1费远程,若你上回合使用了预知则+4伤害) + **命运编织** (通用,看牌顶3张排序)
- **既视感** — 每回合第一张打出的"你上回合也打出过的同名牌"费用 -1
- **时间线收束** (传说) — 本回合你可以看到牌堆所有牌的顺序并重新排列前 5 张

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 手牌质量始终最优 | 爆发天花板不高 |
| 几乎不会"卡手" | 操作繁琐（每次预知都要选择） |
| 长期战极强，回合越多越碾压 | 短期战斗（3回合内结束）不如其他流派 |

## 与其他职业的定位区分

- **vs 先锋**：先锋物理叠加直接暴力，灵能者需要通用牌铺垫再释放。灵能者更"策略"，先锋更"战术"。
- **vs 黑客**：灵能者专注伤害增幅，黑客专注 debuff/控制。灵能者是"输出核心"，黑客是"控制核心"。
- **vs 共生体**：灵能者用通用牌"安全"地叠增幅，共生体用 HP "危险"地叠增幅。灵能者更安全，共生体更极端。
