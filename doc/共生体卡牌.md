# 共生体 (Symbiote) — 卡牌设计

> 配合 `共生体.md` 流派设计使用。

---

## 设计说明

- **HP**: 75
- **核心机制**: **侵蚀 (Erosion)** — 消耗 HP 强化卡牌效果；HP < 50% 时触发 **侵蚀共鸣**：全效果 +30%
- **卡池比例**: 近战 40% / 远程 25% / 通用 35%
- **默认站位**: 灵活
- **侵蚀规则**: 侵蚀消耗 HP 不能将 HP 降至 1 以下

### 关键词说明

| 关键词 | 说明 |
|--------|------|
| **侵蚀 X** | 支付 X 点 HP，获得增强效果（不可将 HP 降至 1 以下） |
| **侵蚀共鸣** | 当前 HP < 50%（≤37）时，所有效果 +30% |
| **生物质** | 可堆叠的资源。累计 5 层可消耗：恢复 10 HP 或造成 15 伤害 |
| **寄生** | 施加于敌人的堆叠状态。敌人回合结束时：每层寄生造成等量伤害并恢复你等量 HP，然后寄生层数 -1 |
| **孢子** | 对所有敌人施加中毒效果的 AOE 机制 |
| **进化** | 消耗生物质获得永久增益（持续整场战斗） |
| **裂变** | 根据当前站位（前排/后排）触发不同效果 |

## 附录：设计备注

1. **侵蚀平衡**：侵蚀消耗的 HP 不能将 HP 降至 1 以下，确保玩家不会因侵蚀直接死亡。
2. **生物质经济**：生物质获取途径多样（直接获取、侵蚀产出、失去 HP 触发），消耗途径同样丰富（回血、伤害、进化），需要玩家做出资源分配决策。
3. **寄生与中毒联动**：传说卡「万疫之母」(SY-104) 可实现中毒→寄生的联动，与孢子瘟疫流和寄生宿主流形成跨流派协同。
4. **裂变双子定位**：裂变流需要频繁切换站位，因此提供了多张 0 费切换牌和切换时触发效果的能力牌来保证流畅度。
5. **进化适应的长线价值**：进化流在短期战斗中可能不如其他流派爆发力强，但在多层 Boss 战中因永久增益的叠加而越来越强。
6. **侵蚀共鸣的全局影响**：HP < 50% 时 +30% 全效果是共生体的核心被动，所有流派都可受益，但侵蚀极限流会刻意维持低 HP 以最大化此增益。
