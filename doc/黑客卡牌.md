# 黑客 (Netrunner) — 卡牌设计

> 配合 `黑客.md` 流派设计使用。

---

## 卡牌总览

| 统计项 | 数值 |
|--------|------|
| 卡牌总数 | 105 |
| 初始卡牌 | 10 |
| 普通卡牌 | 35 |
| 罕见卡牌 | 30 |
| 稀有卡牌 | 20 |
| 传说卡牌 | 10 |

**射程分布**

| 射程 | 数量 | 占比 |
|------|------|------|
| ⚔️近战 | 5 | ~5% |
| 🎯远程 | 37 | ~35% |
| ⚙️通用 | 63 | ~60% |

**类型分布**

| 类型 | 数量 |
|------|------|
| 攻击 | 33 |
| 技能 | 42（其中程序子类型 29 张） |
| 能力 | 30 |

**关键词说明**

| 关键词 | 说明 |
|--------|------|
| 协议栈 | 打出"程序"子类型牌时累积；核心植入体决定栈的被动效果 |
| 程序 | 技能牌子类型，标记为 `技能[程序]`；打出时累积协议栈 |
| 入侵 | 使目标敌人攻击其盟友而非你（入侵核心：每 5 层栈触发） |
| 暴露 | 标记状态，受到伤害 +50%（渗透核心：3+ 层栈远程牌施加） |
| 穿透 | 无视目标护甲（渗透核心：每层栈使远程牌穿透前排遮挡） |
| 中毒 | 每回合开始时受到等于层数的伤害，持续扣血 |
| 虚弱 | 造成的伤害降低 25%（可叠加） |
| 脆弱 | 受到的伤害增加 50%（可叠加） |
