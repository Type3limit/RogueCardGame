# 共生体 (Symbiote) — 职业设计与流派指南

> 本文档应位于 `doc/roleDesign/共生体.md`。卡牌详细列表见 `共生体卡牌.md`。

---

## 职业概览

| 属性 | 数值 |
|------|------|
| HP | 75 |
| 核心机制 | **侵蚀 (Erosion)** — 消耗 HP 来增幅卡牌效果，HP 越低某些卡牌越强 |
| 卡池比例 | 近战 40% / 远程 25% / 通用 35% |
| 默认站位 | 灵活（取决于 build） |
| 设计幻想 | 死亡边缘的极限战士 / 以血为食的生物兵器 / 不断进化的完美生命体 |

## 核心机制：侵蚀 (Erosion)

共生体的许多卡牌有"侵蚀"效果——消耗自身 HP 来获得更强的效果（更高伤害、更多护甲、额外效果）。HP 越低，某些被动效果越强。

- 侵蚀是可选的：大部分侵蚀牌可以正常打出（基础效果），也可以"支付"额外HP获得增强效果
- 默认核心（侵蚀核心）：HP 低于 50% 时所有卡牌效果 +30%
- 自残 ≠ 自杀：侵蚀效果不能将 HP 降到 1 以下

**核心张力**：你的 HP 既是生命值也是资源。花多少 HP "投资"到伤害增幅？留多少保命？这个平衡每回合都在变。

## 核心植入体

| 植入体 | 规则改变 | 定义的流派 |
|--------|---------|-----------|
| **侵蚀核心**（默认） | HP < 50% 时所有卡牌效果 +30% | 走钢丝极限型 |
| **再生核心** | 取消侵蚀增幅。失去HP时获得等量"生物质"层数，5层可消耗回复10HP或造成15伤害 | HP资源循环型 |
| **寄生核心** | 伤害不再直接生效，改为施加等量"寄生"层数。敌人回合末寄生→伤害并治疗你等量HP | 吸血持续型 |
| **裂变核心** | 取消侵蚀。前排近战回复2HP；后排远程溅射2全体 | 双形态切换型 |

## 六大构筑流派

### 流派一：侵蚀极限 (Erosion Edge)

**核心组合**：侵蚀核心 + 过载回路
**站位策略**：前排为主（高风险）
**幻想**：在死亡边缘起舞。你的血量越少，你就越强。每一滴流出的血都转化为毁灭的力量。

#### 玩法概述

侵蚀核心在 HP < 50% 时所有效果 +30%。过载回路给 4 能量但未用完自伤——这不是劣势，是**加速降血工具**。你主动自残到 50% 以下，然后享受 +30% 全面增幅，用4能量的高行动力快速结束战斗。

#### 核心策略

- **前2回合**：用侵蚀牌快速降到 50% HP（约 37 HP），激活 +30% 增幅
- **激活后**：4能量 + 30%增幅 = 每张牌都比正常强约 50%
- **风险管理**：携带 1-2 张回复牌作为保险，在即将致命时拉一口血

#### 关键卡牌

- **血怒打击** (1费近战,消耗3HP造成12伤害) + **侵蚀爆发** (消耗5HP,本回合攻击+50%)
- **生命保险** — 被动：你的HP首次降到1时，回复15HP并获得10护甲
- **至暗时刻** (传说) — 将HP降到1，本回合所有卡牌效果 ×3

#### 站位考量

前排风险巨大（低HP+吃所有伤害），但近战牌占40%需要前排。核心决策：什么时候该退后排保命？

#### 优劣势

| 优势 | 劣势 |
|------|------|
| +30%全面增幅=全流派最高综合效率 | 低HP一个失误就死 |
| 过载回路自伤反而帮你激活 | 遇穿透敌人或AOE直接暴毙 |
| 4能量高行动力 | 休息点必须回血，成长选择受限 |

### 流派二：生物质引擎 (Biomass Engine)

**核心组合**：再生核心 + 标准神经接口
**站位策略**：灵活
**幻想**：你的身体是一台完美的生物反应堆。流出的血变成生物质，生物质变成治疗或毁灭——一个永不停止的循环。

#### 玩法概述

再生核心让你每次失去 HP 时获得等量生物质层数。5 层生物质可消耗为：回复 10 HP 或造成 15 伤害。这意味着自残不是"花费"而是"投资"——失去 5HP → 获得 5 生物质 → 回复 10HP = 净赚 5HP。

#### 核心策略

- **循环引擎**：自残 → 获得生物质 → 用生物质回血/伤害 → 自残...
- **生物质管理**：优先储到 10+ 层，然后根据需要选择回血或伤害
- **对敌人伤害也算**：被敌人打也获得生物质，所以站前排吃伤害也是"收入"

#### 关键卡牌

- **生物质吸收** (通用,消耗4HP获得6生物质) + **血肉重塑** (通用,消耗5生物质回复12HP)
- **生物质爆破** — 消耗所有生物质，每 5 层对全体敌人造成 15 伤害
- **完美循环** (传说) — 本回合每次失去HP时获得双倍生物质

#### 站位考量

灵活——前排吃伤害获得更多生物质，后排更安全。根据生物质存量决定：高存量→后排安全输出，低存量→前排"挨打积攒"。

#### 优劣势

| 优势 | 劣势 |
|------|------|
| HP循环可持续性最强 | 输出依赖生物质积攒，启动慢 |
| 被动收入(被打也获得生物质) | 一次性被打太多可能来不及循环 |
| 战斗越长越强(持续积攒) | 不如侵蚀极限的瞬间爆发高 |

### 流派三：寄生宿主 (Parasitic Host)

**核心组合**：寄生核心 + 标准神经接口
**站位策略**：前排为主
**幻想**：你的每一次攻击都在敌人体内种下寄生虫。它们慢慢吞噬宿主，而你靠吸取的养分维持生命。

#### 玩法概述

寄生核心改变伤害方式：你的伤害不再直接生效，而是施加等量"寄生"层数。敌人回合结束时，寄生层数转化为伤害并治疗你等量 HP。这是延迟伤害+吸血的组合。

#### 核心策略

- **大量施加寄生**：每回合用尽可能多的攻击牌给不同敌人挂寄生
- **延迟收割**：你的伤害在敌人回合结束时才结算——这意味着你打完牌后有一个"等待"期
- **前排自持**：寄生回血让你站前排也能自我维持，不需要额外回血手段

#### 关键卡牌

- **寄生注射** (1费近战,施加8层寄生) + **孢子释放** (远程,对全体施加3层寄生)
- **深层寄生** — 寄生转化伤害时，额外施加 2 层新寄生（持续扩散）
- **寄生爆发** (传说) — 立刻引爆所有敌人身上的寄生（不等到回合末），伤害翻倍

#### 站位考量

天然前排——吸血效果让你能承受前排伤害。寄生回血越多，你越能站稳前排。

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 吸血让前排生存极强 | 伤害延迟一回合（敌人回合末才结算） |
| 多目标挂寄生=全体AOE吸血 | 敌人在寄生结算前可能行动（打你） |
| 战斗越长寄生叠越多 | 对需要快速秒杀的场景不利 |

### 流派四：裂变双子 (Fission Twin)

**核心组合**：裂变核心 + 战场接口
**站位策略**：频繁换位（核心）
**幻想**：你有两种形态——前排是疗愈的生命之泉，后排是毁灭的溅射风暴。每次切换都是一次蜕变。

#### 玩法概述

裂变核心给前后排完全不同的被动：前排近战回复 2HP，后排远程溅射 2 全体。战场接口免费换位+减费。你在两种模式间高速切换，前排攒血后排输出。

#### 核心策略

- **前排回血轮**：打 2-3 张近战牌，每张回复 2HP（= 4-6HP回复）
- **后排溅射轮**：切到后排打远程牌，每张对全体造成额外 2 溅射
- **标准节奏**：前排近战(回血) → 换后排(免费) → 远程(溅射) → 下回合前排...

#### 关键卡牌

- **形态切换** (0费通用,换位+抽1牌) + **生命打击** (近战,伤害+回复) + **溅射射线** (远程,AOE)
- **双重形态** — 被动：你每次换位后首张牌效果 +50%
- **完美裂变** (传说) — 本回合你同时拥有前排和后排的被动效果

#### 站位考量

**必须频繁换位**。不换位只能享受一半被动，裂变核心的全部价值在于切换。

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 攻防一体(前排回血后排输出) | 出牌顺序极重要 |
| 溅射提供持续AOE | 单体伤害不突出 |
| 自我回血+灵活站位=高生存 | "换位失甲"环境直接崩盘 |

### 流派五：孢子瘟疫 (Spore Plague)

**核心组合**：侵蚀核心 + 脉冲处理器
**站位策略**：后排为主
**幻想**：你是行走的生化武器。每一次自残都释放致命孢子，在敌群中扩散瘟疫——你越痛苦，敌人越绝望。

#### 玩法概述

侵蚀核心的 +30% 增幅（< 50% HP）配合脉冲处理器的打牌抽牌。核心机制：用自残牌降血激活侵蚀，同时自残牌释放"孢子"效果（中毒/虚弱/AOE），脉冲处理器保证连锁不断。

#### 核心策略

- **降血+施毒并行**：侵蚀牌同时降血和施加中毒/孢子效果
- **脉冲连锁**：低费孢子牌 → 打牌抽牌 → 更多孢子牌 → 全场中毒
- **增幅毒伤**：50%以下HP时，中毒效果也享受 +30% 增幅

#### 关键卡牌

- **毒孢子** (1费远程,消耗2HP对全体施加2中毒) + **瘟疫种子** (0费通用,消耗3HP对目标施加4中毒)
- **疫病扩散** — 一个敌人死亡时，其中毒层数转移到所有存活敌人
- **终末瘟疫** (传说) — 消耗10HP，对所有敌人施加15层中毒+5层虚弱

#### 站位考量

后排为主——你HP不高还要自残，需要后排的安全。但偶尔换前排打近战孢子牌。

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 全场AOE中毒效率极高 | 双重自残(侵蚀+自残牌)极其危险 |
| 增幅毒伤=持续AOE伤害 | 对免疫中毒的敌人完全失效 |
| 脉冲连锁保证牌量 | 前期极脆弱(降血+低HP) |

### 流派六：进化适应 (Adaptive Evolution)

**核心组合**：再生核心 + 预知模块
**站位策略**：灵活
**幻想**：你是完美的进化体。每一次受伤都让你更强。战斗越久，你的身体越完美——直到你成为不可阻挡的存在。

#### 玩法概述

再生核心的生物质积攒 + 预知模块的牌堆操控。这个流派不追求快速结束战斗，而是通过长期积攒"进化"层数（通过特定卡牌将生物质转化为永久增幅），越打越强。

#### 核心策略

- **前期积攒**：用自残牌和被打来积攒生物质，不急于消耗
- **中期进化**：用"进化"类卡牌将生物质转化为永久buff（如永久+2攻击、永久+5HP上限）
- **后期碾压**：多次进化后你的基础属性远超正常值，任何牌都变得很强

#### 关键卡牌

- **基因强化** (通用,消耗5生物质永久+2所有攻击伤害) + **适应进化** (通用,消耗5生物质永久+5最大HP)
- **快速进化** — 被动：你每积攒10层生物质时自动获得1层"进化"(所有牌效果+5%)
- **终极形态** (传说) — 消耗所有生物质，永久获得：每10层生物质=+3攻击+10最大HP+1能量(仅本战斗)

#### 站位考量

灵活——前期多站前排挨打积攒生物质，中后期buff叠高后任意站位都很强。

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 长期战天花板最高(永久buff) | 极其慢热，前5回合近乎无用 |
| 预知模块保证关键牌获取 | 短战斗/速杀场景完全不适合 |
| 越打越强的成长感极佳 | 需要大量"进化"牌支撑，选牌受限 |

## 与其他职业的定位区分

- **vs 先锋**：先锋用外部资源(能量/超载)，共生体用内部资源(HP)。先锋安全可靠，共生体刺激极端。
- **vs 灵能者**：灵能者通过通用牌"安全铺垫"，共生体通过自残"危险投资"。节奏和风险完全不同。
- **vs 黑客**：黑客削弱敌人(控制导向)，共生体强化自己(自我增幅)。黑客更团队，共生体更自我。
