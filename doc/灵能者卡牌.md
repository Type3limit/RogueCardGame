# 灵能者 (Psion) — 卡牌设计

> 配合 `灵能者.md` 流派设计使用。

## 职业概览

- **生命值**：65
- **核心机制**：**共鸣 (Resonance)** — 打出通用/技能牌时积累共鸣层数，增幅攻击伤害
- **卡池比例**：近战 10% / 远程 50% / 通用 40%
- **默认站位**：后排

### 站位规则

| 站位 | 效果 |
|------|------|
| 前排 | 每次获得护甲时额外+3护甲；近战牌可用 |
| 后排 | 每回合第一张远程牌伤害+20%；近战牌不可用 |

### 共鸣机制说明

- 打出 ⚙️通用 或 技能牌时，获得 **1层共鸣**（部分卡牌可获得更多）
- 攻击牌可 **消耗共鸣** 来增幅伤害（具体倍率由卡牌决定）
- 共鸣在战斗结束前持续存在，除非被特定效果消耗或清除
- 基础共鸣增伤：每层共鸣使下一次消耗共鸣的攻击 +2 伤害

### 标记 (Mark)

灵能者的第二核心机制。被标记的敌人受到额外效果：

- 标记默认持续 **1 回合**（回合结束时自动消除）
- 标记**不可叠加**——重复标记只刷新持续时间
- 被标记的敌人受到灵能者攻击时，伤害 **+3**
- 部分卡牌对被标记目标有额外效果（详见各卡牌描述）
