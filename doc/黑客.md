# 黑客 (Netrunner) — 职业设计与流派指南

> 本文档应位于 `doc/roleDesign/黑客.md`。卡牌详细列表见 `黑客卡牌.md`。

---

## 职业概览

| 属性 | 数值 |
|------|------|
| HP | 70 |
| 核心机制 | **协议栈 (Protocol Stack)** — 打出程序牌累积栈层数，栈越高被动效果越强 |
| 卡池比例 | 近战 5% / 远程 35% / 通用 60% |
| 默认站位 | 后排 |
| 设计幻想 | 削弱敌人的debuff大师 / 控制敌人心智的傀儡师 / 编织无限连锁的引擎构筑者 |

## 核心机制：协议栈 (Protocol Stack)

打出"程序"子类型的通用牌时累积协议栈。默认效果（协议核心）：每层栈在回合开始时对随机敌人施加 1 点虚弱。

- 只有标记为"程序"的通用牌才累积协议栈（非全部通用牌）
- 协议栈在战斗内持续存在（不自然消退）
- 不同核心植入体完全改变协议栈的效果

**核心张力**：你花行动打"程序"牌构建协议栈（没有直接伤害/防御），还是打其他牌应对眼前威胁？栈越高被动效果越强，但你为此牺牲了即时行动力。

## 核心植入体

| 植入体 | 规则改变 | 定义的流派 |
|--------|---------|-----------|
| **协议核心**（默认） | 每层栈回合开始对随机敌人施加 1 虚弱 | debuff 压制型 |
| **入侵核心** | 每 5 层栈入侵一个敌人（攻击盟友而非你），入侵后栈清零 | 心智控制型 |
| **编织核心** | 取消协议栈。每抽一牌 → 下张通用牌 -1 费。0费牌抽1牌（每回合限3次） | 引擎连锁型 |
| **渗透核心** | 每层栈让远程牌无视前排遮挡。3层+ 远程牌额外施加暴露（+50%伤害） | 穿透狙杀型 |

## 六大构筑流派

### 流派一：协议压制 (Protocol Suppression)

**核心组合**：协议核心 + 标准神经接口
**站位策略**：常驻后排
**幻想**：你是战场上的病毒。每个回合敌人都变得更弱、更慢、更笨——直到他们连还手之力都没有。

#### 玩法概述

经典黑客玩法。通过打程序牌叠协议栈，栈层数对应回合开始的虚弱数量。高栈时敌人全部被虚弱，伤害大幅降低。同时用远程牌和debuff牌持续输出。

#### 核心策略

- **前 2 回合**：优先打程序牌到 3-4 层栈（回合开始自动施加虚弱于 3-4 个目标）
- **中期维持**：保持 5+ 层栈，敌人持续处于虚弱，全队承受伤害大减
- **输出**：依赖弱化后的"暴露协议"等牌——对已虚弱目标额外伤害

#### 关键卡牌

- **数据注入** (1费程序,+1栈+小伤害) + **系统扫描** (程序,+1栈+看敌人意图)
- **弱点分析** — 对已虚弱敌人造成的伤害 +50%
- **零日攻击** (传说) — 消耗所有协议栈，对所有敌人施加等量虚弱层数

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 全队生存大幅提升(多人极佳) | 自身输出不高，依赖队友/时间 |
| 虚弱是无条件防御——任何伤害都减少 | 对免疫虚弱的Boss完全失效 |
| 后排安全，极少承受伤害 | 前几回合栈低时队伍很脆弱 |

### 流派二：神经劫持 (Neural Hijack)

**核心组合**：入侵核心 + 标准神经接口
**站位策略**：后排
**幻想**：你不需要打败敌人——你只需要让他们互相残杀。傀儡师站在暗处，操纵战场上的每一个敌人。

#### 玩法概述

入侵核心让协议栈每满 5 层就"入侵"一个敌人（该敌人本回合攻击盟友）。核心循环：快速叠 5 层栈 → 入侵 → 栈清零 → 再叠 → 再入侵。每次入侵等于让一个敌人跳反。

#### 核心策略

- **速叠 5 层**：最快 2 回合叠到 5 层（每回合 2-3 张程序牌）
- **入侵目标选择**：优先入侵高伤害敌人（让它打自己队友）
- **循环入侵**：叠5→入侵→叠5→入侵，中后期几乎每 2 回合入侵一次

#### 关键卡牌

- **后门植入** (1费程序,+2栈) + **快速编译** (0费程序,+1栈) + **入侵脉冲** (强制入侵不需5层但费用高)
- **深度渗透** — 被入侵的敌人攻击力翻倍（打自己队友时伤害更高）
- **全面控制** (传说) — 立刻入侵所有敌人（不消耗栈），持续 1 回合

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 每次入侵=删除一个敌人的行动+造成敌方内部伤害 | 栈清零后有2回合空窗期 |
| 对多敌人场景极其强力 | 单个Boss无法入侵（没有盟友可打） |
| 不需要自己输出伤害 | 免疫控制的敌人完全无效 |

### 流派三：无限编织 (Infinite Weave)

**核心组合**：编织核心 + 脉冲处理器
**站位策略**：后排
**幻想**：你不是在打牌——你是在编写代码。每抽一张牌都让系统更快，直到整副牌在一个回合内被执行完毕。

#### 玩法概述

编织核心取消协议栈，改为抽牌引擎：每抽 1 牌 → 下张通用牌 -1 费；0 费牌抽 1 牌（每回合限 3 次）。配合脉冲处理器（打牌抽牌），形成"抽牌→减费→0费→抽牌"的连锁。

#### 核心策略

- **牌组极度精简**：控制牌组在 15-18 张，确保连锁不断裂
- **大量 0-1 费通用牌**：每张都可能触发减费→0费→抽牌的循环
- **伤害收割**：连锁中穿插远程攻击牌（被减费后0-1费），大量出牌=大量伤害

#### 关键卡牌

- **数据碎片** (0费通用) + **快速编译** (0费程序) + **数据脉冲** (1费远程)
- **递归算法** — 打出这张牌时若手牌>3张，抽2牌；否则从弃牌堆取回1张0费牌
- **无限循环** (传说) — 本回合编织核心的抽牌限制取消（从3次变为无限次）

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 成型后单回合行动次数全游戏最多(10-15张) | build成型极慢，前期极弱 |
| 每张牌都很便宜，能量效率最高 | 牌组膨胀=连锁断裂，选牌需极度克制 |
| 和"数据节点"环境极度契合 | 每回合限3次抽牌限制了天花板 |

### 流派四：幽灵狙击 (Ghost Snipe)

**核心组合**：渗透核心 + 预知模块
**站位策略**：后排
**幻想**：穿透一切防线，直接命中最重要的目标。你是数字幽灵，没有人能躲过你的精确打击。

#### 玩法概述

渗透核心让协议栈层数使远程牌无视前排遮挡（穿透），3层以上还额外施加"暴露"（+50%伤害）。预知模块确保你总能找到高伤害远程牌。

#### 核心策略

- **快速到 3 层栈**：2 回合内叠到 3 层，解锁穿透+暴露
- **优先击杀后排**：穿透直接打后排治疗/强化者/脆皮高伤害目标
- **暴露叠加**：对暴露目标集中火力，+50% 伤害让你快速收割

#### 关键卡牌

- **精准打击** (2费远程高伤害) + **数据注入** (程序叠栈) + **目标锁定** (远程,对暴露目标+3伤害)
- **穿甲协议** — 你的远程牌对暴露目标额外无视护甲
- **幽灵猎杀** (传说) — 选择一个后排敌人，对其造成 25 伤害并标记（下回合所有队友对其+50%）

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 直接打后排高价值目标 | 栈不够3层时穿透力不足 |
| 暴露让团队总伤害大增 | 输出集中在单目标，AOE弱 |
| 预知保证每回合有高质量手牌 | 对全前排敌人配置，穿透无意义 |

### 流派五：病毒矩阵 (Virus Matrix)

**核心组合**：协议核心 + 过载回路
**站位策略**：后排
**幻想**：释放数字瘟疫。你不直接杀敌，你让病毒在敌群中扩散——虚弱、中毒、数据腐蚀，直到他们自己崩溃。

#### 玩法概述

协议核心的虚弱效果 + 过载回路的 4 能量 = 更快叠栈 + 更多 debuff 牌。4 能量让你每回合比标准多打 1 张牌，加速协议栈构建。同时用"病毒"类远程牌施加持续伤害(DOT)。

#### 核心策略

- **双线并行**：程序牌叠栈(被动虚弱) + 病毒牌施加DOT(主动伤害)
- **能量精确**：4能量要全部用完——2程序+2病毒/远程 是标准配比
- **后期爆发**：高栈(6+)时虚弱削减敌人伤害，病毒持续掉血，敌人自行消耗

#### 关键卡牌

- **病毒注入** (1费远程,施加2层中毒) + **数据注入** (1费程序) + **蠕虫传播** (远程,中毒扩散到相邻敌人)
- **恶意代码** — 对已虚弱的敌人，你的中毒效果伤害翻倍
- **逻辑炸弹** (传说) — 对所有敌人施加"延时炸弹"(3回合后爆炸造成30伤害,期间每层虚弱+5爆炸伤害)

#### 优劣势

| 优势 | 劣势 |
|------|------|
| debuff+DOT双线压制 | 没有即时高伤害，慢热型 |
| 4能量加速叠栈和DOT | 自伤风险(70HP不高) |
| 全体虚弱+全体中毒=AOE强 | 对免疫debuff的敌人完全失效 |

### 流派六：冰墙构筑 (ICE Architect)

**核心组合**：编织核心 + 战场接口
**站位策略**：前后灵活切换
**幻想**：你在自己周围构建数字防火墙——"ICE"程序。每一道防线都是一个反击陷阱。敌人每攻击你一次都会付出代价。

#### 玩法概述

编织核心的抽牌引擎 + 战场接口的免费换位。你用连锁抽牌获得大量手牌，同时在前后排切换触发减费和换位效果。防御型黑客：用"ICE"牌（能力牌/通用牌）建立被动防御。

#### 核心策略

- **铺设ICE**：前几回合打出多张"ICE"能力牌（持久防御效果）
- **换位触发**：战场接口换位减费 + 编织核心抽牌减费 → 大量低费ICE快速部署
- **反击为主**：ICE 不主动伤害，而是在被攻击时反击/减伤/反弹

#### 关键卡牌

- **防火墙** (1费能力,每回合获得3护甲) + **入侵检测** (能力,被攻击时对攻击者造成4伤害)
- **蜜罐陷阱** — 能力：敌人攻击你时有 30% 概率攻击失效并对自己造成等量伤害
- **全域免疫** (传说) — 本回合免疫所有伤害，回合结束时对所有攻击过你的敌人各造成 15 伤害

#### 优劣势

| 优势 | 劣势 |
|------|------|
| 中后期几乎无敌(ICE叠加) | 前期ICE没铺好时极度脆弱 |
| 被动反击让敌人自己送死 | 输出完全依赖反击,主动伤害近零 |
| 编织引擎保证ICE获取速度 | 对不攻击你的敌人(如DOT/AOE)无效 |

## 与其他职业的定位区分

- **vs 先锋**：先锋正面暴力输出，黑客间接控制+debuff。先锋是"矛"，黑客是"毒"。
- **vs 灵能者**：灵能者增幅自己伤害，黑客削弱敌人防御。效果看似相似，但黑客的debuff惠及全队。
- **vs 共生体**：黑客控制敌人，共生体消耗自己。黑客更安全但更慢，共生体更危险但更快。
