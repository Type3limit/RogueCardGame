## v2 → v3 变更总结

| 被砍系统 | 理由 |
|----------|------|
| 协同链 (Synergy Link) | 信息过载、沟通负担重，多人用站位/状态效果自然协同替代 |
| 入侵/改造 (Hack & Salvage) | 入侵战略价值低（不如直接打死），双货币无意义取舍 |
| 适应性 AI | 方向错误——Roguelike 应让玩家适应敌人，而非反向 |
| 遗物系统 | 与植入体边界模糊，职能已被植入体+事件吸收 |
| PvP 模式 | PvE/PvP 卡牌设计有根本冲突，v1 集中资源做 PvE |
| 10 种环境效果 | 精简为 5 种"规则改变型"，砍掉"剥夺信息/纯随机"类 |
| 3 槽植入体 | 精简为 2 槽 (Neural + Core)，仅保留"规则改变型" |
| 零件货币 | 只保留金币单一货币 |

## v3 保留并强化的系统

| 系统 | 改进 |
|------|------|
| 前后排站位 | 后排禁用近战牌（非-25%减伤）、换位消耗 1 能量、单体攻击只打前排 |
| 植入体 (2 槽) | 仅"规则改变型"，每个真正改变玩法而非堆数值 |
| 多阶段敌人 AI | HP 阈值触发行为切换，替代动态适应 |
| 位置反应型卡牌 | 同一张牌前后排不同效果，强化站位核心轴 |
| 原型核心系统 | 前 3 回合观察行为自动激活，解决首幕无 build identity 问题 |
| 卡牌分支升级 | A/B 两条升级路线，替代植入体固件升级成为中后期成长轴 |
