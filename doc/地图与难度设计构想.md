# 地图与难度系统设计构想

> 本文分析当前地图/难度设计的现状，与杀戮尖塔（Slay the Spire）进行对比，提出差异化方案。

---

## 一、现状分析：我们目前是杀戮尖塔模式吗？

### 当前结构

```
第一幕 — 7 层固定节点（3战斗 + 1精英 + 1商店 + 1事件 + 1休息）→ Boss
第二幕 — 8 层节点 → Boss
第三幕 — 9 层节点 → 最终 Boss
```

### 与杀戮尖塔的对比

| 维度 | 杀戮尖塔 | 我们当前设计 | 评价 |
|------|---------|-------------|------|
| 地图结构 | 分叉路径树，每层多个节点可选 | **线性层列表，每层只有1个节点** | ⚠️ 缺少路径选择——Roguelike最核心的策略层 |
| 幕数 | 3幕 + 隐藏第4幕 | 3幕 | ✅ 一致 |
| 节点类型 | 战斗/精英/商店/休息/事件/宝箱 | 战斗/精英/商店/休息/事件/Boss | ✅ 基本一致，缺宝箱 |
| 路径决策 | 每层选左/中/右路径，提前可见 | 无路径选择 | 🔴 **最大缺陷** |
| 精英分布 | 玩家选择是否绕路/冒险打精英 | 固定1个精英 | ⚠️ 缺少风险/收益选择 |
| 难度曲线 | 攀升系统（Ascension 1-20） | 未设计 | 🔴 缺少重玩驱动力 |
| Boss奖励 | Boss遗物三选一 | 核心植入体三选一 | ✅ 对应 |

### 核心问题

1. **线性地图 = 零路径策略** — 当前设计每幕就是"走完N层打Boss"，没有分叉、没有选择、没有风险评估。这丢掉了Roguelike地图50%的乐趣。
2. **精英是强制的** — 玩家无法选择"这局血量低要绕开精英"还是"构筑强要多打精英赚奖励"。
3. **没有难度系统** — 通关后没有更高难度挑战，重玩动力弱。
4. **节点数量偏少** — 第一幕7层太短，构筑感不足。杀戮尖塔每幕约15层。

## 二、地图系统重设计：分叉路径 + 赛博朋克风格

### 2.1 核心概念：网络拓扑地图

**不做杀戮尖塔的"树状路径"，做"网络拓扑"——节点和连线组成的网络图。**

设计理念：游戏的赛博朋克背景下，地图不是"爬塔"，而是在废弃的巴别塔网络中**黑入不同子系统节点**。每个节点是一个子网，连线是数据通道。

```
     ⚔️ ─── ❓ ─── ⚔️
    / \              |
起点    💀 ─── 🏕️ ── ⚔️ ── 🔮 Boss
    \       /        |
     ⚔️ ── 🛒 ─── 💀
```

### 2.2 地图生成规则

#### 每幕结构

| 幕 | 总层数 | 路径宽度（每层节点数） | 精英数 | 商店 | 休息 | 事件 | Boss |
|----|--------|---------------------|--------|------|------|------|------|
| 第一幕 | 12-14层 | 2-4个/层 | 2 | 1-2 | 1-2 | 2-3 | 1 |
| 第二幕 | 13-15层 | 2-4个/层 | 2-3 | 1-2 | 1-2 | 2-3 | 1 |
| 第三幕 | 14-16层 | 3-4个/层 | 3 | 1-2 | 2 | 2-3 | 1 |

#### 路径生成算法要点

1. **每层2-4个节点**，随机生成连线（保证每个节点至少1条入边+1条出边）
2. **第1层必是普通战斗**（教学/热身）
3. **精英节点不出现在前3层**（给玩家构筑时间）
4. **商店节点不出现在第1层和最后2层**
5. **Boss前最后1层强制休息点**（杀戮尖塔经典设计——给你最后喘息机会）
6. **连线可以跨层**（允许"快速通道"——跳过1层但可能错过奖励）
7. **地图生成时就确定所有节点类型，玩家可以看到全图**（信息公开，策略透明）

### 2.3 新增节点类型

除了基础的6种节点（战斗/精英/商店/休息/事件/Boss），增加2种赛博朋克特色节点：

#### 📡 数据终端（Data Terminal）

```
发现一个废弃的数据终端。你可以尝试接入...

[破译加密档案] → 获得1张稀有卡预览 + 在下一次选卡时该卡必定出现
[下载战术数据] → 揭示当前幕所有精英和Boss的行动模式
[抽取能量残余] → 立即获得50信用币，但下一场战斗开始时少1点能量
[拔线离开]     → 无事发生
```

**设计意图**：信息型节点。让玩家用小代价换取情报，强化"有规划的冒险"感。

#### ⚡ 超频挑战（Overclocked Challenge）

```
异常的能量涌入使这个区域不稳定...

战斗规则变化：
- 你的能量每回合+1（从3→4）
- 但敌人全部获得 [狂暴]：伤害+30%
- 胜利奖励：选1张牌 + 1次免费升级
```

**设计意图**：高风险高回报的特殊战斗节点。类似杀戮尖塔的"燃烧精英"但更可控——你提前知道规则变化，可以决策是否进入。

## 三、路径选择的核心策略

### 3.1 风险-收益图谱

路径选择应该让玩家每层都面临**有意义的取舍**：

```
安全路径：⚔️ → 🏕️ → ⚔️ → 🛒     （低风险，稳健成长）
贪婪路径：💀 → ⚔️ → 💀 → ⚡      （高风险，快速强化）
经济路径：⚔️ → 🛒 → ❓ → 🏕️      （商店导向，花钱解决问题）
情报路径：📡 → ⚔️ → 📡 → 🏕️      （信息优势，精准决策）
```

### 3.2 路径记忆与元策略

玩家应该在地图选择中发展出"个人风格"：

- **激进型玩家**：尽量多打精英，追求植入体 + 稀有牌
- **稳健型玩家**：避开精英，多休息多升级，打磨现有牌组
- **商人型玩家**：优先商店节点，通过购买定向构筑
- **赌徒型玩家**：选超频挑战和高风险事件，博取超额奖励

**关键设计原则：没有"最优路径"。不同构筑强度、HP状态、金币数量下，最佳路径不同。**

## 四、事件系统：叙事选择 + 赛博朋克世界观

### 4.1 事件设计原则

1. **每个选项都应该有明确的代价和收益**（不是"选好的还是选坏的"，而是"你愿意付哪种代价换哪种收益"）
2. **部分事件应该和职业/植入体产生联动**（如果你是黑客，某些事件有专属选项）
3. **事件应该创造"这局专属的故事"**（高记忆点）
4. **绝对不做"纯惩罚无选择"的事件**（玩家被坑的感觉在Roguelike中极度劝退）

### 4.2 事件示例（每幕6-8个事件池，随机抽取）

#### 第一幕事件池

**「废弃实验室」**
```
你发现一间被遗弃的神经研究实验室。实验设备仍在运行...

[接受实验注射] → 随机获得一个神经植入体，但永久失去5点最大HP
[拆解设备变卖] → 获得80信用币
[搜索研究日志] → 获得1张随机罕见卡
[小心离开]     → 无事发生
```

**「流浪商人」**
```
一个戴面具的流浪者拦住你的去路。

"我有好东西...但我不收信用币。"

[交出牌组中1张牌] → 获得1张比它高一级稀有度的随机卡
[交出当前50%的信用币] → 随机获得一瓶稀有药水 + 恢复15%HP
[交出1个神经植入体] → 获得1个核心植入体（二选一）—— ⚠️ 仅第二幕后可触发
[拒绝交易] → "那就...小心前路。" 揭示下3层地图节点
```

**「幽灵信号」**
```
你的神经接口突然接收到一段加密信号...

[解密信号] → 需要通过一场特殊战斗（3个弱敌，但你只有2能量）
  成功 → 获得1张传说卡（本幕池中随机）
  失败 → 失去25%当前HP
[屏蔽信号] → 获得10护甲持续到下场战斗
[反向追踪] → 仅黑客专属选项：自动成功，获得传说卡 + 50信用币
```

**「记忆碎片」**
```
你踩到一块破碎的神经芯片，一段记忆涌入你的意识...

[接受记忆] → 获得一张其他职业的稀有卡（每局限1次跨职业牌）
[抵抗记忆] → 获得3层你的职业核心资源（超载/共鸣/协议栈/生物质）持续到下场战斗
[分析芯片] → 获得50信用币 + 1次免费卡牌移除
```

**「黑市接口」**
```
一扇隐蔽的门后是一个地下改造工坊...

[非法改造] → 选择1张牌获得强化升级（效果比普通升级强50%），但该牌每次使用时你受到2点伤害
[购买违禁药水] → 花费40信用币，获得"兴奋剂"药水（使用后本回合+2能量但下回合-1能量）
[举报位置] → 获得100信用币（出卖黑市位置）—— 但后续该幕不再出现商店节点
```

#### 第二/三幕增加更高风险事件

**「植入体过载」（第二幕专属）**
```
你的核心植入体发出异常警报...

[强制超频] → 核心植入体效果翻倍，但每回合开始时失去3HP
[安全重启] → 核心植入体效果降低50%，持续3场战斗，之后恢复正常
[紧急替换] → 立即获得一个新的核心植入体（三选一，替换当前的）
```

**「时间裂隙」（第三幕专属）**
```
空间在这里扭曲了...你看到了另一个时间线的自己。

[与镜像交换] → 随机替换你牌组中3张牌为同等稀有度的其他卡
[吸收镜像] → 永久+10最大HP，但手牌上限-1
[对抗镜像] → 特殊Boss战（镜像使用你的牌组！），胜利后复制你牌组中最好的1张牌（获得第二份）
```

### 4.3 职业专属事件触发

部分事件根据你的职业提供**专属第四选项**：

| 职业 | 触发条件 | 专属选项示例 |
|------|---------|-------------|
| 先锋 | 遇到军事遗迹 | "以军衔验证身份" → 免费获得1张先锋稀有卡 |
| 灵能者 | 遇到精神干扰 | "用念力化解" → 获得3层共鸣 + 预览Boss技能 |
| 黑客 | 遇到电子设备 | "入侵系统" → 获得额外情报/信用币 |
| 共生体 | 遇到生物样本 | "吸收基因" → 永久+5HP + 1层生物质每场战斗开始 |

## 五、难度系统：攀升协议（Ascension Protocol）

### 5.1 概念

通关后解锁**攀升协议**系统——每级增加一个（累计的）难度修正。共20级。

这不是杀戮尖塔的简单数值叠加——我们利用前后排/站位系统创造**独特的难度维度**。

### 5.2 攀升协议等级

| 等级 | 名称 | 效果 | 设计意图 |
|------|------|------|---------|
| 1 | 数据加密 | 精英敌人+10%HP | 基础数值压力 |
| 2 | 补给中断 | 普通战斗金币奖励-20% | 经济压力，迫使选择 |
| 3 | 损坏的治疗模块 | 休息点回复从30%降至20% | 资源更紧张 |
| 4 | 前线加固 | 敌人前排获得+2初始护甲 | 近战职业压力↑ |
| 5 | 狙击协议 | 30%的战斗中会有1个"狙击"敌人无视前排直击后排 | 后排不再安全 |
| 6 | 通胀 | 商店价格+15% | 经济路径削弱 |
| 7 | 侦察干扰 | 地图只显示相邻2层（而非全图） | 路径规划难度↑ |
| 8 | 精英强化 | 精英敌人额外获得1个随机增益（狂暴/再生/护甲翻倍三选一） | 精英风险↑ |
| 9 | 站位压制 | 敌人每3回合强制你换位一次 | 站位策略颠覆 |
| 10 | **诅咒之卡** | 每幕开始时牌组中加入1张"系统错误"诅咒牌（占手牌位，无法使用，需花费金币移除） | 牌组管理压力 |
| 11 | 能量不稳 | 每场战斗有20%概率第一回合只有2能量 | 开局不稳定 |
| 12 | 后排威胁 | 50%的精英战中Boss式后排敌人出现在前排阵亡后 | 后排远程流压力 |
| 13 | 植入体干扰 | 核心植入体三选一变为二选一 | 构筑灵活性降低 |
| 14 | 增援协议 | 精英战中第4回合起每2回合召唤1个小兵 | DPS检测 |
| 15 | **时间压力** | 每场战斗有回合限制（普通20回合，精英15回合，Boss25回合）——超时后敌人伤害每回合+50% | 防龟策略 |
| 16 | 腐蚀地图 | 25%的非战斗节点变成"腐蚀节点"（普通战斗+环境效果） | 安全节点减少 |
| 17 | Boss进化 | Boss获得第4阶段 | Boss更难 |
| 18 | 双精英 | 每幕1个精英节点变为"双精英"——同时面对2组精英 | 极端挑战 |
| 19 | 死亡之墙 | 第一幕Boss前的最后休息点消失 | 无喘息 |
| 20 | **完美协议** | 以上所有效果 + 初始HP-10% + 无法使用药水回复超过50%HP | 终极挑战 |

### 5.3 为什么这比杀戮尖塔的攀升更有趣

杀戮尖塔的攀升大部分是数值叠加（+HP、-回复、+伤害），到后期就是"数字更大"。

我们的攀升系统利用了**站位系统**创造独特的机制挑战：
- **攀升5/12**：打破"后排=安全"的假设
- **攀升9**：强制换位让固定站位流派失效
- **攀升15**：时间压力让龟甲流无法无限苟
- **攀升7**：信息限制让路径规划本身变成挑战

**每一级攀升都应该让你重新思考构筑策略，而不只是"打更痛的怪"。**

## 六、让地图更有趣的额外点子

### 6.1 分支剧情线

每幕的地图中隐藏一条**支线路径**（需要满足特定条件才能看到）：

- **第一幕**：如果你在前3层战斗中全程站前排，第4层出现隐藏节点"旧战场"→ 打一场高难战斗，奖励一张专属的"老兵"系列卡
- **第二幕**：如果你持有的未升级卡≤3张，出现"工匠大师"→ 免费升级3张牌
- **第三幕**：如果你全程没死（HP从未归0被药水救起），出现"完美主义者"→ 获得独特传说卡

**设计意图**：给老玩家发现的乐趣，增加重玩时的"啊我要试试触发那个隐藏路线"动力。

### 6.2 地图天气系统

每幕的地图有一个**全局环境主题**，影响所有战斗节点（不是单场的环境效果，而是整幕的基调）：

| 主题 | 效果 | 对策略的影响 |
|------|------|-------------|
| 🌐 数据风暴 | 所有战斗开始时随机1张手牌费用变为0 | 高费卡构筑受益 |
| 🔋 能量潮汐 | 奇数回合4能量，偶数回合2能量 | 节奏管理变得重要 |
| 🦠 瘟疫扩散 | 每场战斗开始时你有2层中毒 | 自愈/共生体受益，脆皮职业受压 |
| ⚙️ 机械增殖 | 敌人死亡时有30%概率产生一个1HP的碎片（0伤害但占位） | AOE清场更有价值 |
| 🌑 信号干扰 | 无法预览敌人意图的第一个回合 | 第一回合需要保守出牌 |

每局每幕的天气随机，但**在地图界面提前显示**——你可以根据天气调整路径选择和构筑方向。

### 6.3 NPC系统：旅途中的伙伴

在地图探索中，特定事件可以让你获得**临时NPC伙伴**（持续3-5场战斗）：

- **修理无人机**：每场战斗开始时给你5护甲
- **数据精灵**：每回合额外抽1张牌，但抽到的第6张牌自动弃掉
- **流浪猫（？）**：纯装饰，但如果你带它走完整幕，在Boss前它变成一张稀有卡

**设计意图**：给一局run增加"同行者"的情感联结。Roguelike容易感觉孤独和重复，一个小伙伴能提供意外的归属感。流浪猫设计尤其会创造"我要保护它"的玩家自发目标。

### 6.4 地图互动：留下痕迹

如果未来做多人/社区功能——在单人模式的地图上显示其他玩家的"幽灵数据"：

- "82%的玩家选择了左侧路径"
- "这个精英击杀了47%的挑战者"
- "上一个到达这里的玩家使用了灵能者"

类似《黑暗之魂》的留言系统，但以数据统计形式呈现——契合赛博朋克世界观。

### 6.5 Boss前的抉择之门

每幕Boss前增加一个**特殊节点——"抉择之门"**：

```
通往 Boss 的大门前，你必须做出选择：

🗡️ [激进协议] → Boss战中你的伤害+20%，但你的最大HP临时-15%
🛡️ [防御协议] → Boss战中你每回合+3护甲，但你的能量-1
📊 [情报协议] → 预览Boss全部技能和阶段转换条件，无任何加成/惩罚
⚡ [超频协议] → Boss战中你的能量+1，但Boss所有攻击+25%
```

**设计意图**：在Boss战前给玩家最后一次战略决策。不同构筑在不同协议下有不同的最优选——DPS构筑选激进、防御构筑选超频、不了解Boss选情报。

## 七、与已有系统的整合

### 7.1 环境效果 × 地图

当前的5种环境效果（等离子风暴、重力异常、低温区域、过热区域、数据节点）应该：

1. **与地图可见** — 在地图节点上标注该战斗的环境效果（如果有）
2. **与天气关联** — 某些地图天气会增加特定环境效果的出现概率
3. **精英/Boss固定环境** — 精英和Boss战斗固定搭配特定环境，提前可见，成为路径选择的考量因素

### 7.2 植入体系统 × 地图

- 植入体获取节点在地图上标注（精英战图标旁加植入体符号）
- 数据终端可以预览即将到来的精英掉落哪种植入体（让你提前规划是否要走那条路打精英）

### 7.3 商店 × 地图

- 不同位置的商店有**不同库存倾向**（靠近精英的商店更多药水，靠近休息的商店更多升级素材）
- 每幕有一个"黑市商店"（隐藏路径），物品更好但更贵

## 八、总结：核心差异化

### 我们不是"又一个杀戮尖塔"

| 杀戮尖塔 | 我们 |
|---------|------|
| 爬塔幻想 | 黑入网络拓扑幻想 |
| 静态树状路径 | 动态网络拓扑（含跨层快速通道） |
| 环境效果极少 | 丰富的环境效果 + 地图天气系统 |
| 攀升纯数值 | 攀升利用站位系统创造机制挑战 |
| 无职业专属事件 | 职业专属事件选项 |
| 无NPC同行 | 临时NPC伙伴系统 |
| Boss前无特殊选择 | 抉择之门机制 |

### 优先级排序

| 优先级 | 功能 | 理由 |
|--------|------|------|
| 🔴 P0 | 分叉路径地图 | 没有路径选择就不是Roguelike |
| 🔴 P0 | 攀升协议（至少10级） | 通关后重玩驱动力 |
| 🟡 P1 | 事件系统（每幕6个） | 叙事深度和构筑多样性 |
| 🟡 P1 | Boss前抉择之门 | 低开发成本、高策略价值 |
| 🟢 P2 | 数据终端节点 | 情报系统有特色但非必须 |
| 🟢 P2 | 地图天气系统 | 增加变化但开发量较大 |
| 🔵 P3 | NPC伙伴 | 情感设计，可后期加 |
| 🔵 P3 | 幽灵数据/社区功能 | 需要服务器，最后做 |
| 🔵 P3 | 隐藏支线路径 | 内容量大，属于"内容更新"范畴 |

---

> **设计信条：好的Roguelike地图让每一步都是一个有意义的决策。如果玩家在地图上不需要思考——只是机械地走到下一个节点——那地图系统就失败了。我们要让路径选择和战斗选择一样令人纠结。**
